Carte de visite une pharmacie.

Description du projet.

J'ai réalisé une carte de visite pour une petite pharmacie à partir de leur charte graphique puis j'ai utilisé un mock-up pour leur présenter.

Le but de ce projet était d'apprendre à créer une carte de visite mais aussi d'apprendre à comprendre les besoins et envies d'un client.
